## 1.不能用可变对象作为默认参数的值
```python
def func(numbers=[],num=1):
    numbers.append(num)
    return numbers

print(func())
print(func())

>>> 
[1]
[1, 1] #数据重复
>>> 
```
正确写法
```python
def func(numbers=None,num=1):
    if numbers is None:
        numbers = [num]
    else:
        numbers.append(num)
    return numbers

print(func())
print(func())

>>> 
[1]
[1]
>>> 
```
